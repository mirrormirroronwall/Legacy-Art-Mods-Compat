# Legacy-Art-Mods-Compat
DOL 0.5.9新版图片名兼容工具

Legacy-Art-Mods-Compat.zip为Mod，加载后可自动兼容通过Mod管理器加载的旧版命名美化Mod。图片显示可能存在延迟。如果美化图片显示为原版，请尝试关闭游戏重新打开

DOL-IMG-Batch-Renamer.exe用于批量重命名，**使用前请备份img文件夹**
请放在和img文件夹同一目录下运行（杀毒软件可能会报毒）
<img width="313" height="96" alt="image" src="https://github.com/user-attachments/assets/9ce25c2b-1565-47d0-88c9-e5741f96ec7e" />
（如需使用Paperdoll+，请把旧版的img/body/basenoarms-classic.png放进img/body文件夹）

未覆盖小图标等部分图片

如文件夹中同时存在两种命名规则的图片（如full.png和full_gray.png）将优先使用符合新版规则的（理应优先用full_gray，但我改不好）

非原版且使用hold以外动作的手持物品无法自动处理

可二改二传
